# dundun — 赚钱主线作战板（v0）

> 核心原则：**赚钱优先 + 安全优先 + 快速上线验证 + 以 PR 交付**。

## 0. 主线目标（North Star）
- 把「出海做网站」副业做成主业：可持续收入增长，最终支持你辞职。

## 1. 约束（必须遵守）
- 不直接上线/发布任何对外内容：所有改动以 **PR** 形式交付给 jasper 审核测试后合并。
- 信息安全优先：不保存/不提交任何凭据（账号密码、token、cookies、私钥、API keys）。

## 2. 站点策略（优先级）
1) **AI 工具站（SEO + 免费工具引流）**
2) **轻量 SaaS（订阅）**
3) 游戏站 / 内容站（当具备强流量入口或可快速规模化时）

## 3. 选题机制（低成本、高胜率）
### 3.1 新词新站（你当前的训练营打法）
- 目标：找到“需求真实 + 竞争弱 + 可快速做出 MVP 工具/页面”的关键词簇。

### 3.2 我建议的筛选漏斗（可复制）
- Step A：收集候选关键词（100–300）
- Step B：打分（0–5）
  - 真实需求（搜索意图明确）
  - 可做工具化（能做在线工具/生成器/计算器）
  - 竞争强度（SERP 是否被大站垄断）
  - 变现路径（广告 RPM / 订阅 / leads）
  - 生产成本（能否 1–3 天上线 MVP）
- Step C：挑 Top 1–3 做「1周冲刺」

## 4. 上线节奏（默认）
- Day 0–1：站点骨架 + SEO 基建 + 核心落地页
- Day 2–3：第一个可用工具（免费）+ CTA（订阅/邮件）
- Day 4–7：批量落地页（程序化/模板化）+ 内链

## 5. SEO 基建清单（必须项）
- robots.txt / sitemap.xml
- canonical / metadata / OG / Twitter
- JSON-LD（SoftwareApplication / WebSite / FAQ 等）
- 性能：LCP/CLS 控制、图片优化
- 站内结构：关键词簇 → 目录/标签 → 详情页

## 6. 变现路径（先后顺序）
- 先 SEO 获取流量 → 再广告（RPM）
- 同时埋订阅入口：
  - 免费工具 = 入口
  - Pro 功能/额度 = 订阅

## 7. 我需要你补充的信息（最小集）
- 目标市场（EN/US vs EN/Global）是否更偏向美国？
- 你是否愿意先用 **纯内容/静态落地页 + 轻量交互** 的方式快速铺量？
- 你偏好：先做 1 个大站，还是并行做多个小站？
