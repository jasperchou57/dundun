# Keyword dossier

把你从任何关键词工具里看到的数据（搜索量/KD/CPC/意图）统一记录到 `keyword-dossier.csv`。

## 为什么要做这个
- 防止“凭感觉选题”
- 让我们能复用数据，快速筛出更赚钱的词簇

## 怎么填（最简）
必填：
- keyword
- market（US / Global / CN 等）
- intent（tool / guide / commercial / comparison）
- page_type（准备做成什么页面）

可选：
- monthly_search, kd, cpc
- tool_key（如果对应某个工具页）
- source（截图/工具名）

填完我会：
- 归类成“关键词簇”
- 输出页面规划（主页+长尾页+FAQ+内链）
